Setting up in Canva — step by step
Do this once to create a brand kit in Canva. Every future infographic will automatically have the right colours and fonts available.
1
Create a Brand Kit
In Canva, go to Brand Hub (left sidebar) → Create a Brand Kit → name it "Anurag Mohapatra." This stores your colours and fonts so they appear in every design automatically.
2
Add brand colours
In the Brand Kit, click Add a colour → paste each hex code one at a time and give it a name. Add all eight colours from the Canva Hex Codes table above.
Navy: #0a1628  ·  Gold: #c9a84c  ·  Off-white: #f8f7f4  ·  Ink: #1a1a18
3
Set brand fonts
In the Brand Kit, under Fonts → set Heading font to Playfair Display, Body font to DM Sans. Both are free and searchable directly in Canva's font picker.
4
Upload the AI Spy logo
In Brand Kit → Logos → upload the AI Spy PNG (the hand-drawn logo). This makes it drag-and-droppable in any design. Upload both a white version and a gold version if available.
5
Create a master template
Design one infographic that you're happy with, then save it as a template in Canva. Future infographics start from this template — consistent layout, colours, and fonts guaranteed.
6
Add your branding footer
Every infographic should have a small footer with your name and website. Lock this element in the template so it appears on every design automatically.

Do’s and don’ts
Common mistakes to avoid when creating branded content.
DO — always
Use Navy as the primary background on dark designs
Use Gold only as an accent — one or two elements per design
Use Playfair Display for the main headline only
Keep stat numbers large — they are the centrepiece
Add anuragmohapatra.com as a footer on every infographic
Keep designs clean with generous white/navy space
Export at 2× resolution for crisp display on retina screens
DON’T — avoid
Don’t use more than 2 colours in a single design
Don’t use Gold as a background — it’s an accent only
Don’t mix Playfair and DM Sans at the same size
Don’t use Arial, Roboto, or Canva default fonts
Don’t add drop shadows or gradients
Don’t use purple, teal, or other off-brand colours
Don’t publish without the anuragmohapatra.com footer
Voice and tone for infographics
What the writing inside your infographics should sound like.
Principle
Guidance
Data first
Lead with the number or finding. The headline IS the insight — not a description of what the infographic shows.
Practitioner voice
Write for a fraud professional, not a general audience. Use industry terminology correctly — SAR, APP fraud, mule account, EFTA, PSR.
One idea per infographic
Each infographic should make exactly one argument or reveal one key insight. Not a summary of everything you know on a topic.
Source everything
Every statistic needs a source in the footer. Builds credibility. Format: Source: NICE Actimize Fraud Insights 2025 — 9px, muted grey.
